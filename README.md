# Informatikai_projektfeladat
Laptop kamera segítségével egy arcfelismerő és regisztráló rendszer, ami tevékenységek felismerésére és elemzésére képes.

Tartalomjegyzék

Bevezetés

1.1 A projekt célja
1.2 A rendszer jelentősége és alkalmazási területei

Elméleti háttér

2.1 Arcfelismerés alapjai
2.2 Képfeldolgozás és számítógépes látás (Computer Vision)
2.3 Gépi tanulás szerepe az arcfelismerésben
2.4 Tevékenységfelismerés alapjai

Felhasznált technológiák

3.1 Programozási nyelv: Python
3.2 Könyvtárak és eszközök (pl. OpenCV, face_recognition, TensorFlow/PyTorch)
3.3 Fejlesztői környezet

A rendszer felépítése

4.1 Hardver komponensek (laptop kamera)
4.2 Szoftver architektúra
4.3 Adatfeldolgozási folyamat

Működés részletes leírása

5.1 Arcok detektálása
5.2 Arcok felismerése és regisztrációja
5.3 Tevékenységek felismerése
5.4 Adatok tárolása és kezelése

Megvalósítás

6.1 A program felépítése
6.2 Fontosabb algoritmusok ismertetése
6.3 Kódrészletek bemutatása

Tesztelés és eredmények

7.1 Tesztelési környezet
7.2 Eredmények kiértékelése
7.3 Hibák és korlátok

Továbbfejlesztési lehetőségek

8.1 Pontosság növelése
8.2 Új funkciók bevezetése

Irodalomjegyzék
Mellékletek
9.1 Teljes forráskód
9.2 Képernyőképek
9.3 Egyéb dokumentáció
